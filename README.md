**Goal**

This is an attempt at building an R package for round-trip editing between Quarto documents and Microsoft Word.
The goal was to enable collaborative review workflows while maintaining reproducible document generation for clinical research.

**The Chatbot Coding Experiment**

This code was written in collaboration with a Claude AI chatbot, before coding agents were available.
The design was worked out through discussions, then Claude generated code, which I would review and manually copy-pasted into R Studio
It was my first learning exercise in the limits of AI context windows.
Initially, the code was solid, but quality declined as the chat thread grew longer.
In the end, Claude was confused. I was confused. 
Claude started reintroducing errors that had previously been debugged. Claude made random changes silently that broke things that had previously worked.
Claude had me writing code from scratch for functionality that already exists off the shelf.

**Next steps**

The problem I was trying to solve remains unsolved, but I have hope.
My intent is to redo this project from scratch using a coding agent inside an IDE. Stay tuned!
